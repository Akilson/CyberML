# CyberML

## Aperçu

Vous trouverez toutes les analyses effectuées sur le dataset [`CIC IoT-DIAD 2024`](https://www.unb.ca/cic/datasets/iot-diad-2024.html). Dans le dossier `notebook/` se trouve l'analyse
exploratoire du dataset ainsi que des applications concrètes d'algorithme de
Machine Learning supervisé et non-supervisé pour de la détection et de la classification
d'anomalies.

Dans le dossier `doc/` se trouve le rapport de notre analyse.
